# 枯乃葉のタスク管理 (Windows)

バックグラウンド（トレイ常駐）で動作し、進捗を視覚的に把握できるシンプルなタスク管理ツールです。ついでにメモも書けます。今後機能追加予定。

## 主な機能
- トレイ常駐（閉じる/最小化でバックグラウンド動作）
- 期限と進捗を指定してタスクを追加
- タスクごとの進捗バー表示
- タスクの完了/復元
- メモの追加
- タイマーの開始/停止で作業時間を累積
- 完了したタスクは「完了済み」タブに移動
- 「・・・」メニューから復元/削除が可能
- `%LOCALAPPDATA%\TaskManager\tasks.json` に自動保存

## 実行
```bash
dotnet build "タスク管理.sln"
dotnet run
```

## 補足
- 進捗は一覧の「進捗」列を直接編集できます（0〜100）。
- 「タイマー」列の開始/停止で作業時間が累積されます。
- 「・・・」メニューから完了/復元/削除を操作できます。
- トレイアイコンをダブルクリックすると画面を復帰できます。
